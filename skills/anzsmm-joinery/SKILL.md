## §30 JOINERY

**Items given:**
- (a) Preparation of shop drawings
- (b) Samples

**Key rules:**
- **M1**: Measure separately each type of material or species and grade of timber.
- **M2**: For materials/finishes other than timber, state type, quality, finish — group as CFC, particle board, composite board, similar.
- **M3**: Describe appearance grade timber required to be selected and kept clean for clear finish.
- **M4**: Describe work tongued, housed, glued or secret fixed.
- **M5**: Describe work fixed by any method other than nailing — type/size/spacing.
- **M6**: Describe labours with relevant classification item.
- **M7**: Measure and describe separately timbers impregnated under pressure.
- **M8**: Describe joinery items primed or prefinished before delivery to site.

**Classification & units:**

| Class | Units | Notes |
|---|---|---|
| 1. Joinery fittings (lockers, cabinets, shelving units, bench units, bench tops, similar) | no | State type, overall size, refer to drawings; including material type and all hardware; including forming tap/sink openings. Bench tops may be described and included with the bench unit |
| 2. Shelving, seats, etc (shelves, divisions, seats, ladder frames, legs, gallows brackets, similar) | no | State type, overall dimensions, refer to drawings; state size; whether plain/T&G/slatted; including edge strips; describe component parts |
| 3. Fittings to special design or detail (cupboards, benches, splashbacks, bookcases, tables, counters, blackboards, pinboards, similar) | no | State type, overall dimensions, refer to drawings; including all hardware. **Splashback rules:** if part of bench/counter, include in description of bench/counter; if not integral, measure separately |

**Deemed included (P1):** All labours in framing, notching and fitting around projections, pipes, light fittings, hatches, grilles, similar — complete with cleats, packers, wedges, similar; all nails and screws.

**D1**: Labours such as rebating, grooving, chamfering, splaying, bevelling, tapering, bullnosing, moulding and similar.

## §31 FURNITURE, FITTINGS & EQUIPMENT

**Items given:**
- (a) Preparation of shop drawings
- (b) Samples

**Key rules:**
- **M1**: State type of materials, method of fabrication, capacity where appropriate.
- **M2**: Describe work finish required.
- **M3**: Include manufacturer's name and serial numbers where available.
- **M4**: Measure hospital beds separately.
- **M5**: Measure blackout curtains separately.
- **M6**: Measure motorised systems separately.
- **M7**: Give appropriate unit of measurement for the work involved.
- **M8**: Where artworks to be hung or fitted — describe method of fixing and include any fixings.

**Classification:**

| Class | Units | Notes |
|---|---|---|
| 1. Proprietary furniture (lockers, cabinets, shelving units, compactus, bench units, cupboards, benches, bookcases, tables, counters, desks, work stations, chairs, sofas/lounges, beds, counters, free-standing light fittings, similar) | no | State type, overall size, refer to drawings; including all hardware; state load capacity where specified; include length of base; include proprietary partitions to work stations where same manufacturer |
| 2. Equipment (medical / kitchen / media-AV / sports / vending / special theatre and studio / miscellaneous) | no | State type, refer to drawings/spec |
| 3. Curtains and blinds (window curtains/blinds, theatrical curtains, medical screens / bed curtains, miscellaneous) | m² / no | State type, overall dimensions, refer to drawings/spec; describe closing mechanisms; including hanging rods/rails. M5: blackout separately. M6: motorised separately. M7: appropriate unit |
| 4. Soft furnishing (manchester) | no | State no. of sets and nominate size (king, etc.); refer to drawings/spec |
| 5. Miscellaneous furniture and fittings (items not covered elsewhere) | no | State type, overall dimensions, refer to drawings/spec |
| 6. Artwork (artworks) | no | State type, overall dimensions, refer to drawings/spec. M8: describe fixing method, include any fixings |

## When to apply this skill

Apply when measuring or pricing: custom-built joinery (kitchens, vanities, robes, fitted shelving, reception counters, cabinetry, lockers, splashbacks); proprietary furniture (modular shelving, desks, work stations, chairs, lockers, beds, lounges); medical/kitchen/AV/sports/vending/theatre/studio equipment; window/medical/theatrical curtains and blinds (including blackout and motorised); manchester (soft furnishing); artwork.
