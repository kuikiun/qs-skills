# ANZSMM 2018 — Finishes (§26–29)

## Strict application — no inferring or guessing

When applying this skill to QS work:

- Apply **only** rules explicitly stated in this skill or in the cited source standard.
- Do **not** infer, interpolate, or guess rules that aren't explicit.
- If a situation isn't covered by an explicit rule, **stop and flag it** to the user — don't assume by analogy, "common practice," or what seems obvious.
- Quote the rule number (e.g. `M3`, `P1`, `D2`) when applying a rule so the user can verify against the source.
- If contract documents conflict with the standard, contract documents prevail (per the standard's own override rules) — but state the deviation explicitly in the BOQ / cost plan / cost analysis output for the user's review.
- For anything not covered by an explicit rule, refer the user to the source PDF and ask which rule to apply — do not make one up.

This directive applies to every skill in qs-skills.

## §26 APPLIED FINISHES, RENDER AND TEXTURED FINISHES

**Items given:** (a) Provision of samples or special models.

**Common rules (apply across §26):**
- **M1**: Measure area to be covered (i.e. area **before** finishing).
- **M2**: Measure external and internal work separately.
- **M3**: Measure integrally coloured and waterproofed work separately.
- **M4**: Measure work to existing surfaces and in repairs separately.
- **M5**: Measure isolated surfaces ≤1m² separately.
- **M6**: Describe work to a pattern or in more than one colour.
- **M7**: Measure work to surfaces ≤250mm wide separately (except where caused by openings); state total length in each description.
- **M8**: Measure each different type or kind of finish separately.
- **M9**: State composition and mix, total thickness, no. of coats, nature of base, nature of surface finish.
- **M10**: Measure treatment of base separately.

**Definitions:**
- **D1**: Base = concrete, brickwork, blockwork, expanded metal lathing.
- **D2**: Finish = wood/rubber floated, steel trowelled.
- **D3**: Treatment of base = hacking, applying bonding agent, dubbing out existing walls.

**Deemed included (P1):** Temporary grounds, screeds, similar; bedding, grouting, pointing materials; cutting and/or making good around pipes, sanitary fixtures, similar; labour to internal angles; curing of finished work; cleaning down and polishing; covering, protecting and masking surfaces.

### §26.1 In-Situ Applied Finishes

**Key rules:**
- **M1**: Where suspended ceilings without cornices/cover trims — measure 50mm above finished ceiling level for work to walls/columns/vertical surfaces (unless shown >50mm).
- **M2**: **No deductions** for: angles finished in different material ≤100mm girth; grounds behind timber skirtings/architraves; pipe casings ≤250mm wide; panels of plaster ≤1m² finished differently from main body.

**Classification & units:**

| Class | Units | Notes |
|---|---|---|
| 1. Walls and ceilings (walls, battering walls, walls above roof, parapet backs, ceilings, sloping ceilings, coffered ceilings, sloping/flewing soffits of stairs, margins surrounding other materials) | m² | State height in 1m increments where finished soffit >3.5m above floor |
| 2. Columns / 3. piers / 4. beams (panelled work — isolated/attached) | m² / m / no | M3: appropriate unit per panel type |
| 5. Structural steel framing (columns/beams/rafters/similar — angles, channels, UB, hollow sections) | m² | M4: girthed area to be covered. State type and size |
| 6. Cornices / mouldings (cornices, string courses, architraves, ceiling ribs, mouldings, enrichments to panelling) | m | State size; including undercut moulds, raking/vertical |
| 7. Sills and copings (plastering to external sills, copings, similar) | m | State dimensions or girth; including weathered surfaces |
| 8. Metal trims (external angle, internal angle, edges) | m | State size |
| 9. Existing walls — making good openings (existing walls, new filling to existing openings, making good around broken-out openings, where walls/similar removed, down to new tiled dadoes) | m² / m / no | M5: measure 0.5m beyond each jamb/sill of opening |

**Deemed included (P1, additional):** Arrises, rounded/splayed external angles, fair edges, rounded/splayed edges, intersections to groins; quirks, grooves, V-joints next to frames or junctions with other finishes; forming flush joints with other finishes in same plane.

### §26.2 Plasterboard, Fibrous Plaster, Plaster Acoustic Tiles and Similar

**M1**: Measure per §26.1.
**M2**: Where cornices or trimmings other than fibrous plaster are used — make a note at start of fibrous plaster work.

**Classification:**

| Class | Units | Notes |
|---|---|---|
| 1. Walls and ceilings | m² | State thickness, kind of sheets/tiles, fixing/jointing method, supporting timber/metalwork (type, size, spacing, fixing), hanger length, structure to which fixed |
| 2. Beams and column casings (beams / columns) | m | State no. and width of sides and soffits; including external salient angles requiring pre-formed angles; including shaft cap and base. Also separately work fixed direct to structures without supporting timber/metalwork (state structure and member spacing) |
| 3. Columns, pilasters (incl. casings to stanchions and similar) | no | |
| 4. Cornices and mouldings (cornices, other mouldings) | m | State size; including undercut moulds, raking/vertical; if fixed around plasterboard ceilings. M3: extreme length |
| 5. Stopping internal angle between ceiling and wall | m | M4: where no cornice specified |
| 6. Metal trims (external/internal angles, edges) | m | State size |
| 7. Arches, domes, etc | no | |
| 8. Vent faces | no | State whether flush or projecting |

**D1**: Supporting timber/metalwork = battens, grounds, runners, brackets, hangers.
**Deemed included (additional):** External salient angles; edges set and finished flush with solid plastering in same plane; cutting and making good around sprinkler heads and similar.

### §26.3 Metal Lathing

**M1**: Measure per §26.1.
**M2**: Work fixed direct to structures without supporting timbers/metalwork — separate; state structure and member spacing.

| Class | Units | Notes |
|---|---|---|
| 1. Walls / 2. ceilings / 3. similar | m² | State type, thickness, mesh, laps, fixing of lathing; type/size/spacing of supporting timber/metalwork; hanger length; structure to which fixed |
| Raking / circular cutting | m | |

**D1**: Supporting timber/metalwork = battens, grounds, runners, brackets, hangers.

## §28 CARPET AND RESILIENT FINISHES

Same general rules as §27 (common finishes rules M1–M8).

**Definitions:**
- D1: Base = concrete, timber, sheet flooring.

## §29 PAINTING

**Items given:** (a) Provision of sample finishes, colours, similar.

**Note:** Cross-references where priming or painting is measured in other Sections.

**Key rules:**
- **M1**: Measure area to be covered; make appropriate allowances for additional area on edges, carved/panelled/similar surfaces.
- **M2**: Each type of finish separately.
- **M3**: State nature of base, preparatory work, no. of priming/sealing coats, no. of undercoats, no. of finishing coats, surface finish.
- **M4**: Describe and state material applied by particular method.
- **M5**: External and internal work separately.
- **M6**: Parti-coloured and picked work separately.
- **M7**: Describe work on carved, panelled and similar surfaces.
- **M8**: Surfaces ≤250mm wide or girth of different specification — measure separately in m.
- **M9**: Add area of work to returns and reveals at attached piers, beams and openings to general area.
- **M10/M11**: Group doors in one description; flat overall, both sides; describe.
- **M12/M13**: Group windows in one description; flat overall, both sides; describe.
- **M14**: Roller shutters/grilles/grates/frames — flat overall, both sides; describe.
- **M15**: Where picked out in colours or different finish from surrounding work — measure separately.
- **M16**: Net length for borders.
- **M17**: Polishing per painting classifications.

**Classification:**

| Class | Units | Notes |
|---|---|---|
| 1. Walls | m² | Describe if open grille type screen walls |
| 2. Ceilings | m² | State height where finished soffit >3.5m above floor; state if sloping |
| 3. Doors (doors / frames, linings, architraves, trims / glazed doors) | m² | M10: group in one description |
| 4. Windows (windows / borrowed lights / glazed screens / similar / reveal linings, sill boards, architraves) | m² | M12: group in one description |
| 5. Roller shutters and grilles (roller shutters, roller grilles, grilles, gratings, frames, similar) | m² | M14: flat overall, both sides |
| 6. Balustrades, railings, fencing (solid / open — balustrades, fencing, gates, railings) | m² | State spacing and size of members |
| 7. Roof trusses (open web joists, trusses) / 8. lattice work / 9. similar | m² | |
| 10. Fascias / 11. barges / 12. purlins / 13. pelmets / 14. cornices / 15. picture or dado rails / 16. skirtings / 17. handrails / 18. cappings / 19. gutters / 20. pipes / 21. conduits / 22. flashings / 23. similar | m | |
| 24. Rainwater heads / 25. access manholes / 26. hatches / 27. ladders / 28. boxes / 29. small carved surfaces / 30. rosettes / 31. similar | no | State size |
| 32. Writing (letters, figures, punctuations, similar) | no | State height and style |
| 33. Paper, vinyl and fabric hanging (walls, ceilings, dadoes, infilling to panels) | m² | State widths of rolls and borders, manufacturer's catalogue/reference no.; describe mode of hanging, preparatory work, lining paper; describe stripping/stopping/cutting out cracks/repairing existing plastering |
| Borders / borders cut to shape | m | M16: net length |
| 34. Polishing | per painting classifications | M17 |

**Deemed included (P1):** Provision, erection and removal of scaffolding; preparation, rubbing down between coats; protection/masking of floors, fittings, similar; removing and replacing door/window furniture; removing splash and spray marks.

## When to apply this skill

Apply when measuring or pricing: render, plaster, textured finishes; plasterboard, fibrous plaster, plaster acoustic tiles; metal lathing; ceramic/stone/composite tiling; slab and paving (interior or exterior); carpet, vinyl, resilient sheet; painting on any surface (walls, ceilings, doors, windows, joinery, metal, structural); paper, vinyl and fabric hanging; polishing of timber; staircases and trim finishes; making good existing finishes.
