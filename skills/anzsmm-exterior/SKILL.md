# ANZSMM 2018 — Exterior & Included Sums (§37, §39)

Inherits all general rules from `anzsmm-general`. Covers external/site works (landscaping and roadworks) plus the BOQ summary section for included sums.

---

## §37 EXTERIOR ELEMENTS

### §37.1 LANDSCAPING

**Key rules:**
- **M1**: Topsoil per §4.2 Filling and Hardcore.
- **M2**: Surfaces >30° from horizontal — measure separately.
- **M3**: Maintenance of landscaping works → per the appropriate Sections concerned.

**D1**: Treatment of the ground surface prior to placing topsoil.

**Classification:**

| Class | Units | Notes |
|---|---|---|
| 1. Topsoil (planting boxes / beds) | per §4.2 | |
| 2. Cultivation treatment | m² | M2: surfaces >30° from horizontal separately |
| 3. Seeding / 4. turfing / 5. mulching | m² | Including associated fertiliser; establishment by watering, weeding, mowing, rolling if required |
| 6. Trees / 7. shrubs / 8. similar | no | State type and size; including planting |
| 9. Maintenance of landscaping works | item | M3 |
| 10. Irrigation installation | item | |
| 11. Timber fencing | m | State height, sizes and spacing of posts/rails/pickets/similar; depth and number of post holes; preservatives to base of posts |
| Doors / gates / special framing / strutted posts / special posts / similar | no | State size |
| 12. Wire mesh screens / 13. metal fences / 14. similar | m | State height, mesh type/thickness, support height/spacing; supports, wire strands, selvedged edges, similar |
| Doors / gates / strutted posts / special posts / similar | no | Including hangings, fastenings; state size |

### §37.2 ROADWORKS

**Items given:**
- (a) Masking and protection of adjacent surfaces from bitumen spray and splashes.

**Key rules:**
- **M1**: Filling, hardcore, road base and similar — measure **after consolidation**.
- **M2**: Measure per §4.2 Filling and Hardcore.
- **M3**: Where filling and base course materials are <250mm thick, base course and sub-base may be described and included within the m² item for slabs ≤150mm thick.
- **M4**: Surface treatment to roads/carparks in m².
- **M5**: Asphalt and bitumen surfaces separately, stating depth.
- **M6**: Treatment to existing surfaces separately.
- **M7**: Per §6 Concrete, keeping base materials separate.
- **M8**: For slabs <150mm thick, optionally measure in m² — including base/sub-base materials (each ≤250mm thick) and all finishes; describe and measure thickenings/integral foundation beams in m.
- **M9**: Describe kerb/gutter including concrete, formwork, reinforcing steel and any integral finish. Measure coloured kerbs/gutters separately.
- **M10**: Line marking to roads/carparks — separately.
- **M11**: Describe and enumerate road symbols (e.g. disabled parking, cycle lane).
- **M12**: Describe and enumerate letters to road signs.

**Classification:**

| Class | Units | Notes |
|---|---|---|
| 1. Filling | per §4.2 | |
| 2. Hardcore | per §4.2 | |
| 3. Road surfacing | m² | M4–M6 |
| 4. External concrete pavings | m³ or m² | M7 (per §6 Concrete) or M8 (optionally m² where ≤150mm thick) |
| 5. Concrete roads | m³ or m² | Same as 4 |
| 6. Concrete kerbs, gutters, etc | m | M9 — describe fully |
| 7. Line marking | m | M10. P2: separate by width, spacing, colour |
| 8. Road symbols / lettering | no | M11/M12. P3: separate symbols by size/colour. P4: separate lettering by size/colour |

**Deemed included (P1):** Levelling, grading, trimming, similar in preparing surfaces of filling for road pavements.

## §39 INCLUDED SUMS

This section lists all included sums (Provisional and Contingency) contained in the various sections of the specification — **except Prime Cost (PC) sums** which appear in their respective trade sections.

**Required for each sum:**
- Value of each sum
- Section in which it has been included

**Additional items (where relevant):** Margin, taking delivery, storage, insurance, attendance and fixing.

**Classification:**

| Class | Description |
|---|---|
| 1. Contingency | Sums set aside for unforeseen work (typically a percentage; may be released only on instruction) |
| 2. Provisional | Sums for work that cannot be accurately defined or measured at tender — to be properly identified in BOQ (§2.4) |
| 3. Net or Prime Cost (PC) and the like | Net cost or PC sum for nominated supplies — appears in respective trade sections; not duplicated here |

### How to use in a BOQ

1. **Through the BOQ:** any provisional or contingency sum referenced in a trade section must be cross-listed in the §39 Included Sums section.
2. **PC sums stay** with their trade — don't double-count them in §39.
3. **Adjacent items** for margin / taking delivery / storage / insurance / attendance / fixing → list **here** (§39), not in the trade section, where relevant.
4. **Provisional work** (work that can't be accurately measured per general §2.4) is described in the contract documents and identified in the BOQ — its value rolls up here.

## When to apply this skill

Apply when measuring or pricing: site landscaping (topsoil, cultivation, seeding, turfing, planting, mulching); fencing (timber, wire mesh, metal); gates, posts; irrigation systems; landscape maintenance; roadworks (filling, hardcore, asphalt, bitumen, concrete pavings, roads, kerbs, gutters); line marking, road symbols and signage lettering; **also** when assembling the BOQ Included Sums section (Contingency, Provisional, PC summary), or when deciding where a monetary sum belongs (trade section vs §39).
