# ANZSMM 2018 — Interior (§20–23, §25)

Inherits all general rules from `anzsmm-general`. Covers internal carpentry, partitions, ceilings, access floors, and hardware. Internal doors are in `anzsmm-envelope` (§24).

## §20 CARPENTRY

**Particulars:** Timber sizes (sawn or dressed); unless described as finished, all sizes given are nominal.

**Note:** Composite timber-framed partitions may be measured per §21 Partitions.

**Key rules:**
- **M1**: Measure separately each species, type or grade of timber.
- **M2**: Describe appearance grade timber required to be selected and kept clean for clear finish.
- **M3**: Describe work tongued, housed, glued or secret fixed.
- **M4**: Describe work fixed by any method other than nailing — type/size of fixing, spacing.
- **M5**: Describe labours with relevant classification item.
- **M6**: Measure and describe separately timbers impregnated under pressure (creosote, similar).
- **M7**: Describe joinery items primed or prefinished before delivery.
- **M8**: Group framing in locations (ground floor / upper floor / wall / ceiling / roof / similar).
- **M9**: May measure same-size timbers together stating members.
- **M10**: Nogging, solid bridging, herringbone strutting, similar — measure horizontally over main members where cut between studs/joists.
- **M11**: Nogging/framing for fixing fittings, fixtures, similar — measure separately.
- **M16**: Measure m where ≤250mm wide (except where caused by openings).
- **M17**: Measure each side (boarding).
- **M18–M20**: Linings — m if ≤250mm wide; panelling areas ≤1m² by no.; reveals of openings separately; panelling on rake or in spandrels separately.
- **M22**: Caps to columns and pilasters.
- **M23–M25**: Standard windows per §18; non-standard windows and louvres in description; standard louvres per §18.
- **M26**: Standard ready-made fittings per §31 Furniture.
- **M27/M28**: Bolts following items of work to be fixed; may give in description.

## §21 PARTITIONS

**Items given:** (a) shop drawings, (b) samples, (c) tests.

**Common rules:**
- **M1**: Composite partitions include timber or metal framing.
- Always measure metal vs timber framing **separately**.
- Always measure separately where there are special design requirements.
- Doors/hatches/air registers/similar in openings → state size; measure with the partition concerned (M3, M9, M12, M15).

**Classification:**

## §23 ACCESS FLOORS

**Items given:** (a) shop drawings, (b) samples.

- **M1**: Measure m where ≤250mm wide (except where caused by openings).
- **M2**: Measure **after** floor systems in which they occur.

**Classification:**

| Class | Units | Notes |
|---|---|---|
| 1. Access floor system | m² | Describe grid or gridless, particulars of pedestals, grid type, panel type/size, level tolerance requirements |
| Raking / circular cutting | m | Including fitting |
| Access panels / hatches / ramps / similar | no | State size |
| Trimming around isolated/attached columns and similar | — | |

## §25 HARDWARE

**Items given:** (a) Master keying, keying alike, construction keying or similar systems for keying of locks (state all requirements).

**Key rules:**
- **M1**: Describe type of fixings.
- **M2**: State nature of background to which hardware is fixed.
- **M3**: Describe finish required.
- **M4**: Group separately window, door and sundry hardware.

**Classification:**

| Class | Units | Notes |
|---|---|---|
| 1. Locks / 2. latches / 3. closers / 4. push plates / 5. pull handles / 6. bolts / 7. kick plates / 8. similar | no | State type, size where appropriate; including furniture |
| 9. Fanlight and window sash geared openers | no | Describe fixing; state cable/conduit lengths, no. of windows operated, height above floor of operating gear and sash bottom rails |

**Deemed included (P1):** Matching fixings.
**D1:** Fixings such as bolts, power fixings, other metal anchors, fastenings and similar.

## When to apply this skill

Apply when measuring or pricing: structural timber framing, trusses, fascias, fillets, battens, insulation, boarding, linings, panelling, casings, skirtings, cornices, dado/picture rails, architraves, mouldings; internal stairs, balustrading, handrails; framed screens, internal windows; partitions (metal/composite/acoustic/fire-rated/modular/cubicle/operable walls); suspended ceilings (grid/coffered/bulkheads); access floors; door and window hardware (locks, latches, closers, hinges).
