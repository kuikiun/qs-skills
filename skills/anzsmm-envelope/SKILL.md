# ANZSMM 2018 — Envelope (§16–19, §24)

Inherits all general rules from `anzsmm-general`. Covers the building envelope: roof, facade, openings (windows, glazing, doors).

---

## §16 ROOFING

**Items given:** (a) Guarantees.

**Key rules:**
- **M1**: Apply equally for similar coverings to walls, awnings, soffits, fascias.
- **M2**: Group according to material under appropriate heading.
- **M3**: Measure **net area to be covered** (incl. under ridges, flashings and similar) unless stated otherwise.
- **M4**: Work to surfaces ≤250mm wide measured separately (except where caused by openings); state total length per description.
- **M8**: Add area of all turned up/down ends of sheeting to general area of roof covering.
- **M9**: Measure over girth of any curved sheeting.
- **M11–M12**: Net length for special items.
- **M13**: Insulation — when between purlins/rafters/similar, measure area over such members and state to that effect.
- **M14**: Turn-ups next to walls and similar — measure separately.
- **M15–M22**: Net length for flashings (M15), separate stepped work (M16); net length over sumps/angles for gutters (M17); net length along centre line of pipes over bends/junctions/offsets for downpipes (M18); to downpipes (M19); to ducts/flues (M20); proprietary fascias net length (M21); separation strips only where specifically detailed in contract documents (M22).

## §18 WINDOWS

**Items given:** (a) shop drawings, (b) samples, (c) tests.

**M1**: Describe primed/painted before delivery.
**M2**: Non-standard windows → per §20 Carpentry.
**M3**: Louvres in doors/windows/screens → measure in description of the particular item.
**M4**: Non-standard louvre blades → per §20 Carpentry.

**Classification & units:**

| Class | Units | Notes |
|---|---|---|
| 1. Metal windows | no | State type/size, refer to drawing/detail. Including glazing, flyscreens |
| 2. Metal louvre frames and louvres (frames / blades / louvred windows) | no / m | Frames: state height, no. of blade sockets, fixed or adjustable. Blades: state no. and max length. Windows: individually pivoted; refer to drawing; including flyscreens |
| 3. Timber windows / 4. timber louvre blades | no / m | D1: standard timber windows and blades. Including glazing, sash balances for double hung |
| 5. Folding glass doors | no / item | Measure doors by size and type (incl. frames) |
| 6. Flashings | m | State girth in 250mm increments |

**Deemed included (P1):** Frames, mullions, transoms, glazing bars and beads, trims, caulking and pointing, sashes, casements, hinges, fastenings and other hardware.

## §19 GLAZING

**Items given:** (a) shop drawings.

**Key rules:**
- **M1**: Describe type and thickness of glazing material and type of framing.
- **M2**: Describe and measure separately glazing of irregular shape — measure to smallest rectangular area from which it can be cut.
- **M3**: Measure in 0.5m² increments to 2.5m², thereafter in 2.5m² increments.
- **M4**: Measure overall for the panel.
- **M5**: Measure above springing.
- **M6**: Plastic sheeting forming roof/wall coverings → per §16 Roofing.
- **M7**: Cross-references where hardware given separately in another Section.
- **M8**: Measure edges except to frameless doors/windows, mirrors and louvres.
- **M9**: Glass blocks → per §12 Masonry.

**Classification & units:**

| Class | Units | Notes |
|---|---|---|
| 1. Lead lights / 2. copper lights (opening lights, fixed lights) | m² | State no. of panes, no. of panels, irregular shape; cames width/section, panel patterns, reinforced cames |
| Shaped heads (trefoils, quatrefoils, similar) | m² | Above springing |
| Saddle bars | m | |
| 3. Double glazed units | m² | State no. of units; describe glazing in both panes. D1: factory-made hermetically sealed |
| 4. South lights and glazing bars (glazing bars) | m | State no. of panes/bars, height of fixing above ground; type, material, fixings, end stops |
| 5. Applied reflective finishes | m² | State no. of panes; type of material, adhesion method, priming, edge treatment |
| 6. Acrylic and polycarbonate sheeting | m² | D2: sheeting other than applied finishes |
| 7. All glazing material assemblies (walls/screens) | no | State overall size, refer to drawings; describe work — no. of fixed panels, fin supports, doors, all fixings between panels and to structure |
| 8. Louvre blades | m | State width and no. of blades; treatments to edges |
| 9. Mirrors / 10. panes of silvered glass | no | State width and height; describe backing, frame, holes, brilliant cutting, surface finish, edge processing, colouring |
| 11. Face processing (treatment, finger slots, holes, similar) | no | State type and thickness of glazing material |
| 12. Edge processing (straight / circular edges / edges of bent panes / scallops / internal mitres) | m / no | State type and thickness, no. of panes, bevel width if applicable, edge treatment |
| 13. Silicone jointing and similar | m | State joint width, glazing material; type of joint |
| 14. Glass blocks | per §12 Masonry | |

**Deemed included (P1):** Beading.

**Items given:** (a) shop drawings, (b) samples, (c) tests.

| Class | Units | Notes |
|---|---|---|

## When to apply this skill

Apply when measuring or pricing: roof coverings (membrane, asphalt, sheet, slate, shingle, tile); cappings, ridges, hips, valleys; flashings; gutters and downpipes; sumps; insulation and sarking; safety mesh; skylights, ventilators; facade systems (proprietary, composite, curtain walling); spandrel/infill panels; metal/timber windows, louvres; glazing (single, double-glazed, lead lights, mirrors, silicone jointing); external metal/timber/fire doors, roller shutters, frames.
