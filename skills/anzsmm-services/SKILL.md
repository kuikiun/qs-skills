## §32 HYDRAULICS

**Notes given:**
- (a) Where pipeline drawings are not provided or contract documents don't clearly indicate full extent of all pipelines — state on what basis pipework has been measured.

**Items given:**
- (a) Notices, permits, fees, completion certificates from local authority
- (b) Testing installations
- (c) Sterilizing of installations
- (d) Marking pipes (colour bands, tags, plates, badges, lettering, arrows for flow direction) — give quantity and scope per type of installation

**Key rules:**
- **M1**: Measure separately each different type of installation.
- **M2**: State quality and kind of material in each description / sub-heading.
- **M3**: Describe work surface treated (other than painting).
- **M4**: Painting per §29.
- **M5**: No distinction for fixing to various background surfaces other than for metal/steel — describe and measure those separately.
- **M6**: Work outside boundary of property — measure separately under appropriate sub-heading.
- **M7**: Describe work where under public roadways/footpaths — include any making good.
- **M8**: Measure net length along centre lines of pipes over all bends, junctions, tees, similar fittings.
- **M9**: No distinction for piping use, main stacks vs branches.
- **M10**: Measure fittings to piping ≥32mm nominal diameter.
- **M11**: Measure irrespective of pipe diameter.
- **M12/M17**: Describe fully.
- **M13**: Alternatively describe with traps/cocks/similar.
- **M14**: Except where given in description of another numbered item.
- **M15**: Irrespective of pipe diameter.
- **M16**: Lagging — except where supplied with the pipe.
- **M18**: Tanks etc. — per flat sheet rules in §16 Roofing.
- **M19**: Pits per §33 Drainage.

## §33 DRAINAGE

**Notes given:**
- (a) Where pipeline drawings not provided / extent of pipelines not clear — state basis of measurement.
- (b) Where contract documents don't give invert levels of drains or info needed for depth calculation.

**Items given:**
- (a) Notices/permits/fees/completion certificates
- (b) Testing
- (c) Keeping excavations free from rain/percolating water (state if below water level; tidal levels)

**Key rules:**
- **M1**: Measure separately work in each different type of drainage system.
- **M2**: Work outside boundary of property — separately under appropriate sub-heading.
- **M3**: Describe where under public roadways/footpaths — include any making good.
- **M4**: Where appropriate measure all excavation and subsequent disposal **before excavation**; state classes of material; state commencing level.
- **M5**: Net length along centre lines of pipes over all bends, junctions, similar fittings.
- **M6**: Concrete encasing — include in description of pipe concerned.
- **M7**: Manholes/inspection chambers/septic tanks — refer to spec and drawing details.
- **M8/M9**: Trench width for excavation — for single pipe ≤450mm diameter: pipe diameter + 0.5m; >450mm: twice diameter. For multiple pipes side-by-side ≤450mm overall: overall width + 0.5m; >450mm: twice overall width.
- **M10/M11**: Trench width **in rock** — for single pipe ≤450mm: pipe diameter + 1.25m; >450mm: twice diameter. For multiple pipes ≤450mm overall: twice overall width; >450mm overall: twice overall width + 250mm.
- **M12**: For pits (in rock excavation), measure 250mm beyond plan dimensions, irrespective of depth.

**Definitions:**
- **D1**: Depth = actual excavation depth (depth to invert of pipe + thickness of pipe wall + bedding material under).
- **D2**: Kind of material — concrete, vitrified clay, cast iron, plastic.
- **D3**: Type of backfilling — excavated, sand, gravel, special.
- **D4**: Excavation in rock — extra over excavation in any other class for each type.

**Deemed included (P1):** Maintaining faces of drain pipe trenches and pits, backfilling, disposal of surplus spoil, forming pockets beneath collars, all jointing and other incidental materials.

**Classification:**

| Class | Units | Notes |
|---|---|---|
| 1. Pipework (in trench / suspended / laid in tunnels / fittings) | m / no | Including excavation, depth in 1m increments. State internal diameter, kind of material, bedding, backfilling type, consolidating, levels of backfilling, laying/jointing method. State if vertical. Supports, bends, junctions, inspection bends/junctions, reducers as no. |
| 2. Encasing to pipes (concrete / broken metal/gravel/other surrounds) | m | State mix and thickness of concrete and pipe internal diameter; including additional excavation and formwork. Agricultural pipes — state surrounds dimensions and pipe internal diameter |
| 3. Gullies, traps and accessories / 4. sealing discs / 5. clear outs / 6. drainer's grates / 7. similar (gullies / traps / siphons / standard grease traps / non-return valves / similar) | no | State type and size, internal diameter of inlet/outlet; including grates, sealing plates; method of jointing; setting/surrounding with concrete, formwork, additional excavation |
| 8. Drain connections (existing sumps / inspection chambers / manholes / septic tanks / street channels / existing pipe drains / public mains / similar) | no | State pipe internal diameter and other relevant info |
| 9. Spall or soak away drains | m | State dimensions, type of filling, cover, depth of excavation in 1m increments |
| 10. Manholes / 11. inspection chambers / 12. interceptor traps / 13. septic tanks / 14. similar (in 500mm depth increments) | no | State plan dimension; including excavation, sand/gravel/special backfilling, components |
| 15. Head walls / 16. drain outlets / 17. similar | no | |
| 18. Patent strip agricultural | m | State type and size |
| 19. Excavation in rock (pipe trenches, pits) | m³ | 1m depth, >1m, then 1m increments. State pit size |

## §34 ELECTRICAL INSTALLATIONS

**Items given:**
- (a) Compliance with Acts, By-laws, regulations
- (b) Notices, permits, completion certificates, fees
- (c) Testing installations
- (d) Samples
- (e) Plans, diagrams, manuals, "as fitted" drawings

**Key rules:**
- **M1**: Particulars of materials and workmanship.
- **M2**: Separate work into groups under appropriate sub-headings:
  - (a) Main switchgear, transformers, equipment serving more than one distribution group (LV, MV, HV separately)
  - (b) Power distribution, incoming mains, submains
  - (c) Lighting (d) Heating (e) Mechanical (f) Lift installations (g) Telephones (h) Bells (i) Clocks (j) Sound distribution (k) Signals (l) Fire alarms (m) Burglar alarms (n) Security systems (o) Earthing conductors (p) Lightning conductors (q) TV installations (r) Cable ladders, trays, ducts.
- **M3/M4**: Describe primed/painted before delivery; surface treated other than painting.
- **M5**: Painting per §29.
- **M6**: Describe fixing method, nature of background.
- **M7**: Work to be carried out by Statutory Electrical Authorities — measure separately.
- **M8**: Describe fully.
- **M9**: Cable carrier fittings — measure for ≤150mm wide or high.
- **M10**: Conduits — measure for ≤100mm.
- **M11–M14**: Cables — drawn/laid as net length where in conduits/ducts/trunking or fixed to trays. Other cables as laid (no allowance for sag between boxes/equipment). 0.5m extra on each cable entering switches/outlets/light fittings/similar; 1m extra on each cable entering generators/switchgear/appliances/similar (measure to mid-point of item).
- **M15**: Cable terminations — measure at each cable core or multiple cable core termination.
- **M16**: Fire rated sleeves — measure separately.
- **M17**: Group circuits/sub-circuits separately in each zone, floor area or point of origin and identify.
- **M18**: Proprietary or standard bus ducts/ways separately.
- **M19/M20**: Earthing/lightning conductor cables and terminations — per cable termination rules above.
- **M21–M25**: Describe fully (switchgear, outlets, accessories, light fittings, marking, work to existing services).
- **M26/M27**: Work covered by other Sections — include under appropriate heading; describe where required as part of the electrical installation.
- **M28**: Excavation in rock per §33 Drainage.

**Definitions:**
- **D1**: Switchgear = switchboards and distribution boards.
- **D2**: Components = circuit breakers, isolating switches, change-over switches, switch fuses, fused switches, contactors, starters, meters, fuse chassis (incl. ways per chassis), CTs, test switches, isolating links, bus bars, neutral and earth bars.
- **D3**: Special boxes = adaptable boxes, floor trap boxes other than standard junction boxes.
- **D4**: Joints = except in running length.
- **D5**: Low voltage section = bells, clocks, telephones, fire alarms, burglar alarms, loud speakers, appliances, similar.
- **D6**: Pits for underground electrical services.

**Deemed included (P1):** Screws, nails, bolts, nuts, nipples, standard cable fixing/supporting clips, saddles, brackets, straps, rivets, plugs and similar items and all incidental accessories.

**P2 (laid in trench):** Excavation, maintaining faces, backfilling, compaction, termite treatment where appropriate, cable covers, warning tape, disposal of spoil.

**Deemed included (P3):** All fixings to various building surfaces (electrical conduits, fittings, equipment).
**P4 (switchgear):** Interwiring and connections between components and between components and control gear within the board.
**P5 (cable carriers):** Cutting and jointing cable carriers at fittings, cutting/forming openings in carriers for conduits/entries from equipment, cradles, pinracks for supporting cables, suitable earthing.
**P6 (conduits):** Standard junction boxes, fittings, fixing conduits to/through all building surfaces.
**P7 (cables):** All cable supports and fixings; draw wires, draw cables, similar.
**P8 (cable terminations):** Skinning ends to expose conductors; cable termination glands; insulating/jointing materials; labour to terminals.
**P9 (circuits/sub-circuits):** All items listed for conduits, cables and conduit sleeves; cable joints, skinning ends, connecting to switches/outlets/accessories/light fittings/similar.
**P10 (bus ducts/ways):** All standard supports and brackets.
**P11 (switches, outlets):** Wall boxes and similar.
**P12 (light fittings):** Lamps and diffusers where appropriate.

## §35 MECHANICAL INSTALLATIONS

**Items given:**
- All plant, equipment, ductwork and terminals, pipework, fixtures, fittings, seismic restraints, associated electrical work for the complete system.
- (a) Balancing air conditioning, lift level adjustment, similar
- (b) Testing (state whether fuel/water/electricity supplied free)
- (c) Shop and as-installed drawings/diagrams; instruction manuals
- (d) Specific service and maintenance requirements

**Notes given:**
- (a) Conformance with statutory authorities
- (b) Basis of measurement where installation drawings not provided

**Key rules:**
- **M1**: Each different installation under appropriate heading.
- **M2**: State quality and material in each description / sub-heading.
- **M3/M4**: Describe primed/painted before fixing or delivery; surface treated other than painting.
- **M5**: Painting per §29.

## §36 TRANSPORTATION SERVICES

**Items given:**
- (a) Shop and as-installed drawings/diagrams, instruction manuals
- (b) Specific service and maintenance requirements

**Notes given:**
- (a) Conformance with statutory authorities
- (b) Basis of measurement where installation drawings not provided

**Key rules:**
- **M1**: Each different type of installation under appropriate heading.
- **M2**: State quality and kind of material.
- Internal vs external installations measured separately (1).
- Non-structural surrounds measured separately per appropriate trade.
- Special lift eyes/beams for installation/maintenance — measure separately.

**Classification:**

| Class | Units | Notes |
|---|---|---|
| 1. Escalators | no | State width, total length of horizontal travel and rise; describe balustrade and soffit treatment |
| 2. Travelators | no | State width and total length of horizontal travel; describe balustrade |
| 3. Lifts | no | Describe each lift separately incl. function (passenger/goods); state car size and height travelled, no. of doors |

**Deemed included (P1):** Cooperating with other trades.

## When to apply this skill

Apply when measuring or pricing: cold/hot water plumbing, sanitary plumbing, fixtures (baths/basins/sinks/WCs/urinals/showers); drainage (in-ground pipes, gullies, traps, manholes, septic tanks, soak-aways); rock excavation for drainage; electrical (mains, switchgear, distribution, lighting, low-voltage, communications, alarms, lightning protection, cabling, bus ducts); mechanical (HVAC, ductwork, dampers, grilles, plant, controls); transportation (lifts, escalators, travelators); fire protection (wet sprinklers, dry alarm/detection systems, extinguishers).
