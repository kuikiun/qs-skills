# ANZSMM 2018 — Preliminaries (§2) & Demolition (§3)

Inherits all general rules from `anzsmm-general`. Use this skill when working on Preliminaries or Demolition trade items.

## §2 PRELIMINARIES

### What goes in Preliminaries

Preliminaries describe the project setup, contract conditions, and site conditions. The classification table covers:

| Item | Sub-items |
|---|---|
| 1. Names of parties | Proprietor, architect, all consultants (incl. QS) — names and addresses |
| 2. Description of the site | Visit/inspection arrangements, sand/gravel use/disposal, position, access, document inspection, possession dates |
| 3. Description of works | General description, height above/below ground, number of storeys, restrictions/order of execution |
| 4. Conditions of contract | Form/type, edition, deviations from standard form, special conditions |
| 5. Tender conditions | Particulars of conditions |
| 6. Bills of Quantities | Whether part of contract; QS fee payment method; **state measured per ANZSMM 2018, AIQS/MBA/NZIQS/RMBA** |
| 7. Contingencies | Give in BOQ in same manner as in contract documents |
| 8. General particulars | Working hours, rates and wages, sub-letting, certificates, variations, dates, bonus, liquidated damages, rise and fall, taxes, document handling, discrepancies, payment terms, taxes, notices/permits/fees, liability, bonds, extras, approvals, insurance, programs, QA, maintenance/protection, etc. |
| 2. Site conditions | Setting out, temporary installations, services, screens/fences/hoardings, temporary roads, sheds, samples, testing, advertising, plant/equipment, scaffolding, attendance, protection, cleaning, handing over, defects liability, security, supervision, firefighting, drying, dewatering, works below water level |

### Key rules
- **M5–M7**: Information given except where covered by schedule of insertions, in respect of contract requirements, with reference to relevant general/special conditions / spec clause / other documents.
- **M8**: Give subdivisions where time-related to cover establishment, running and removal costs.

### How to apply
- BOQ Preliminaries section opens with the names of parties and the description of the site & works.
- Always state "measured in accordance with ANZSMM 2018" in the BOQ preface.
- Time-related preliminaries (establishment, running, removal) → break out.
- Anything not covered by the schedule of insertions → give explicitly.

## §3 DEMOLITION

### Items to be given
- (a) Inspection of existing premises
- (b) Measures to protect existing works/premises on site or adjoining properties and occupants
- (c) Provision of temporary hoardings, screens, dust sheets — state areas, lengths, height where possible
- (d) **Credit** for recovered materials
- (e) Disconnecting and sealing off services not documented

### Classification & units

| Class | Units | Notes |
|---|---|---|
| 1. Demolishing whole buildings or major sections of structures | item | State dimensions, number of storeys, nature of construction, asbestos/noxious materials |
| 2. Disconnecting and sealing off services | no | Measure where documented, for each service |
| 3. Removal of footings | m³ | |
| 4. Demolition of building components | m³ or m² or m | Give appropriate unit for the work involved |
| 5. Alteration work | no | Describe alteration work to form new areas, openings and similar |
| 6. Removal of: doors / windows / fittings & fixtures / trusses / similar items | no | State size and whether trims and other items in connection are included |
| 7. Filling openings with new work | — | Measure and give per appropriate Section |
| 8. Making good | — | |
| 9. Trimming around altered work | — | |
| 10. Similar work | — | |
| 11. Removal of asbestos and other noxious materials | item | Compliance with all necessary regulations re notices, protective screens, protective clothing, etc. |
| 12. Designed shoring | item / m² / m | For walls (m²), ceilings (m²), structural steelwork (m), pipes (m), similar items. State approximate cross-sectional dimensions. |
| 13. Salvage or re-fixing | item / no | |
| 14. Approvals | item | |

### Measurement rules (selected)
- **M1**: Give demolition or alteration items with sizes/descriptions conveying nature and extent.
- **M2**: Describe or group alteration works according to location and (if applicable) order of execution.
- **M3**: Describe items to be removed and required to be set aside for refixing.
- **M4**: State where shoring is measured separately.
- **M5**: Measure documented diversions of services and give per appropriate Section.
- **M11**: State where measured separately in other Sections (e.g. items in connection: frames, architraves, sill boards, pelmets, flushing cisterns, taps, pipework).
- **M12**: Measure to existing structures only partially demolished.

### What's deemed included (P1)
- Removing all rubbish and debris
- Making good to work disturbed
- Temporary shoring (other than designed shoring)
- Similar items

## Reminders & gotchas

1. **Credit for recovered materials** — don't forget to give an item; the demolition contractor may claim back salvage value (steel, copper, etc.).
2. **Asbestos & noxious materials** are always a separate item with regulatory compliance — don't bury them in general demolition.
3. **Designed shoring is separate** from temporary shoring (which is included in P1).
4. **Diversions of services** during demolition go under their respective service Sections, not under Demolition.
5. For partial demolition, measure to the structures being kept — protective items (M12).
